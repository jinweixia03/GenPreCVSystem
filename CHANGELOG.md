## [1.0.0] - 2025-03-13

### 新增 ✨

#### CV 任务
- **遥感影像小样本分类 (FSL)** - 基于 Few-Shot Learning 的遥感图像分类功能
- **道路病害检测** - 专用的道路缺陷检测模型支持
- **井盖病害检测** - 专用的井盖损坏情况识别功能
- **图像增强** - 支持亮度、对比度、饱和度、锐度调整
- **图像去噪** - 多种去噪算法支持
- **边缘检测** - Sobel、Canny、Laplacian 三种算法

#### 功能特性
- GPU 自动检测与状态显示
- 文件浏览器使用配置默认目录
- 设置变更后自动刷新文件浏览器
- 增强的帮助文档系统

### 优化 🔧

#### 界面优化
- 简化设置对话框，移除未使用的选项
- 优化使用文档对话框的样式和内容
- 优化快捷键帮助对话框的样式
- 优化关于对话框的样式和内容

#### 代码重构
- 移除 ONNX 依赖和 third_party 目录
- 将 YOLO 重命名为 DL（Deep Learning）全代码库统一
- 重构代码并增强安全性
- 完善代码注释

#### 性能提升
- 优化 GPU 检测逻辑
- 改进批处理性能
- 优化环境扫描缓存机制

### 修复 🐛

- 修复井盖/道路病害检测颜色渲染问题
- 修复批处理问题
- 修复使用原始图片而非显示图片进行推理的问题

## 版本说明

### 版本号格式

`主版本号.次版本号.修订号`

- **主版本号**：重大更新，可能不兼容
- **次版本号**：新增功能，向后兼容
- **修订号**：问题修复，向后兼容

### 标签说明

- ✨ `新增` - 新功能
- 🔧 `优化` - 功能改进
- 🐛 `修复` - Bug 修复
- ⚠️ `废弃` - 即将移除的功能
- 🔥 `移除` - 已移除的功能
- 📚 `文档` - 文档更新

## 如何更新

### 从源码更新

```bash
# 备份自定义模型
cp -r src/resources/models ~/backup_models

# 拉取最新代码
git pull origin master

# 重新编译
cmake --build build --config Release

# 恢复模型
cp -r ~/backup_models/* src/resources/models/
```

### 使用发布版本

1. 下载最新版本的安装包
2. 卸载旧版本（可选）
3. 安装新版本
4. 设置和模型会自动保留
